如何將Table表內資料，修改為以0為首

發表於 2020-02-27 更新於 2021-07-21 分類於 ArcGIS筆記 閱讀次數：

How To Add Leading Zeros To Values In A Field In An Attribute Table?

Solution

步驟1：於TABLE表內新增一欄位(ADD FIELD)為「COUNT_1」，選擇TEXT格式

Step1：Add field named you to want, and select the “text” type.

步驟2：於「COUNT_1」欄位，利用FIELD CALCULATOR重新填入需計算表格的內容

Step2：Select the column you added, use the tool “field calculator” to reset the value.

步驟3：選取「COUNT_1」欄位，操作以下步驟

Step3：Select the column you added,  and operator the following tips.

提示：依資料需求數值之位數，於括號內填入數值

Tip：Specify the desired number of digits in the brackets of .zfill().
