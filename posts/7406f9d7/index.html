一、婚禮當日流程

婚禮儀式加拍照約是90分鐘，以下依儀式流程時間依序介紹

(一)典禮前約10分鐘

引導新人到小房間與牧師共同確認儀式流程，此時新娘父親也會至此稍微彩排後，先行回到禮堂。

(二)儀式開始，約30分鐘

0.行經「迴廊」

此時心情無限緊張，
猶如一開始設計這個「迴廊」的意象一樣，很多回憶會湧上心頭，
深刻知道當走到迴廊盡頭，就是人生另一個幸福與考驗的開始。

1.新人入場

當教堂門一打開，只能用震撼來形容，教堂真的好美，這也是我見到它的第一刻。
蔚藍的天空、遠方的沙灘及海水，其光影交錯所營造出的清透與靜謐感，
兩側牆面的潺潺水聲，一步一步踩著「新人之路」，
三面及地面上的玻璃，與太陽光相互輝映著，教堂內每一個角落都閃耀著光芒。
這一刻的美好與驚豔，深植人心，
整個儀式有種飄飄然，有點不真實的感覺。

1.二次進場

此時親友們可以拿起手機拍照攝影，
二次進場無任何正式儀式，氣氛較為活潑、歡樂，
接下來的時間就是一連串的拍照。

(四)海邊攝影，約20分鐘

即便做了許多簡報與功課，前晚認真練習，
當下依然緊張到一片空白，
很幸運地遇到台籍攝影師引導，找回那些記憶，順利開拍。
婚拍對於害羞、不善拍照的新人真的是一大難題，
前一日開會攝影師也很擔心我會過於緊張，
在拍攝結束時收到稱讚，比他預期表現的好，
也感謝攝影師，拍攝成果張張美麗且美好。

二、感受回饋

愛妮絲渡假婚禮教堂｜五感教堂｜EINES VILLA DE NOZZE OKINAWA
比想像的還美好，完全超出期待，
結婚當天，心情緊張到無所適從，
第一次看見教堂的真實樣貌，
被環境驚艷的同時也被療癒，
這教堂真的有讓人幸福的魔力。

(一)教堂內、外觀及飲食，無可挑剔

  1. 2011年全新落成，2017年4月結婚時，教堂內、外觀與行經之處依然潔淨、透亮，毫無歲月痕跡。
  2. 完全感受到「幸福五感」所創造的美好婚禮，每一個片段都是美好的回憶。
  3. Sunset Bloom餐廳的晚餐十分精緻、美味；早餐也非常豐盛，很可惜沒能完食，擔心吃太多會讓禮服穿起來不好看。

(二)工作人員貼心溫柔又專業

  1. 主要隨行的人員有中文溝通協調員(婚顧)、日籍新娘秘書及攝影師。
  2. Check-in時，婚顧及攝影師即會針對隔日婚禮流程進行說明，並安撫新人們緊張的情緒，強調全然放心置身於儀式中即可；攝影師除會針對事前已準備的簡報做確認外，也會再次安排及確認大合照的分配及站法。
  3. 典禮前90分鐘，日籍新娘秘書開始新娘妝髮服務，婚顧也會隨行在旁，協助溝通。
  4. 日籍新秘會依先前提交簡報資料進行妝髮，妝髮過程非常溫柔，每一個動作也會提前示意，完全重現想要的妝容及髮型，儀式全程跟妝服務，在二次進場前也會詳細確認調整妝容。
  5. 婚顧全程也會幫忙確認事前提到的注意事項，並不斷的提醒新人放鬆，我們真的太過緊張了。
  6. 不會日文的新人完全不用擔心，重要的流程都有婚顧全程協助溝通協調；若會日文，與工作人員的寒暄聊天是這趟婚禮很棒的回憶之一。

(三)婚拍成果，獨特有生命

看過許多 愛妮絲渡假婚禮教堂｜五感教堂｜EINES VILLA DE NOZZE OKINAWA 的婚拍成果，
會根據新人、賓客的個性、互動散發出不同的氛圍，
更特別的是，隨著當下的天氣、光線讓整個教堂有著不同的生命力，
那自然光影、色調的交錯、光芒，是每場婚禮最獨特的禮物，
一樣的教堂，不一樣的婚紗意象。

三、遺憾之處

這無疑是場完美婚禮，
若有機會至 愛妮絲渡假婚禮教堂｜五感教堂｜EINES VILLA DE NOZZE OKINAWA 舉行婚禮的新人們，

  1. 依約定盡早Check-in，享受教堂方專為新人設計提供的Villa Suite 住宿與燭光晚餐，全心沉浸、享受婚禮前的兩人時光。
  2. 沒能寬心享用早餐，覺得有些可惜，但還是會擔心穿禮服會不好看。(超矛盾。)
  3. 真的太過緊張，沒能放心享受一切。(先生說賓客才是最享受的人，確實，舉雙手贊同。)
  4. (希望能有親友舉辦沖繩婚禮，讓我再次感受那份悸動與美好。)

四、其他注意事項提醒

完美婚禮，避免不周之處，提供給大家參考

(一)新人自身體態及膚況的維持與保養

日本妝容及攝影風格崇尚自然、清新，維持自身好狀態以完美婚禮。

  1. 沖繩婚禮的一大誘因是喜愛自然、清新的日本妝容及攝影風格。
  2. 不須擔心日籍新秘的妝髮專業，只需擔心自身狀態不夠好，勤加維護好自身體態及膚況。
  3. 勿過度依賴修圖，避免失去自然攝影風格的特點。
  4. 貼身衣物(UNBRA)須先自行穿戴，在台灣要多練習。
  5. 造型以臉部彩妝及頭髮造型為主，不會上身體水粉等服務，若有需要請自備。

(二)婚前準備溝通項目，越詳盡越好

包括嘉賓名單及前往方式、婚禮儀式細項等，
其中最重要的是造型溝通與攝影溝通，
婚禮代辦公司亦會提供「婚前準備溝通項目」供新人製作，以利後續溝通，
建議在確定婚禮檔期後，就可利用空閒時間進行蒐集，
避免不知所措而花費太多時間，並擁有自己的參考圖庫。

  1. 臉妝及髮型相關參考圖片(可備註彩妝的重點說明)
    日本新秘是會逐一部位呈現的，請避免模擬兩可的空間，極盡可能周詳的製作簡報資料，讓新秘發揮專業的同時，也能讓新人獲得完美妝髮。
    (1)若已有滿意的試妝或訂婚妝容成果提供，最為適合。
    (2)當時製作的部分資料提供參考(圖片取自網路)

  2. 規劃「教堂婚禮攝影」及「沙灘攝影」相關參考POSE的攝影圖片及構圖
    以利後續與攝影師溝通，同時也替自己預習與排練，
    尤其不善拍照的新人，這份作業可以降低許多尷尬及拍攝成果風險，並提升自身的自信。
    (1)若已有確定舉行的婚禮教堂，以該教堂的婚拍成果為主，從中蒐集喜歡的地點、POSE及構圖。
    (2)風格很主觀，無所謂好壞，依自身喜好製作資料即可
    (3)當時製作的部分資料提供參考(圖片取自網路)

(三)親友妝髮造型

建議直接向代辦公司加購，讓長輩避免溝通、通車、遲到等問題。
原先有嘗試找鄰近的美容院協助親友妝髮，
但考慮到親友們無法順暢溝通且有交通上的問題而作罷，
選擇加購是最簡單也最保險的，
而且親友們也都非常滿意日籍化妝師們的妝髮成果。
(親友們的妝髮也需要做細節簡報說明。)

(四)親友接待

教堂方會幫親友們在迎賓廳內準備免費飲料茶水。

  1. 親友直至婚禮儀式開始才會見到新人，若有其他問題或需求請親友相互幫忙。
  2. 親友若為自行前往，請將行李先寄放原住宿飯店或其他親友車上。
  3. 於婚禮前晚再次提醒雙方親友集合時間，請勿遲到。
  4. 若婚禮時段為中午，建議親友晚一點吃早餐後再出發至教堂集合。

(五)最最最重要的是，事前已確認之溝通事項，請勿於婚禮當天臨時變卦，造成新人及工作人員的困擾。
